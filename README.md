# 网管项目

## 目标

- 统一网管,兼具nginx的常用功能与自身配置的灵活

## 规划功能

- 负载均衡
- 灰度策略
- 服务限流
- 路由重写
- 服务熔断
- lua脚本
- 灰度策略(A/B测试)
- 配置热加载
- 黑/白名单
- 全链路日志追踪
- 日志采集的agent(可独立部署)
- 管理后台(可独立部署)
- 核心代理服务(可独立部署)

## 配置持久化存储

- mysql

## 配置热加载(配置中心,支持三种)

- zk
- etcd
-apollo


## 项目结构

|- ginx
    |- doc (各种文档目录)
        |- sql (数据库sql相关文档)
